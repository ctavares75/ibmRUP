Tool Mentor: Adding Comments and Annotations on Shared Models Using Rhapsody Design Manager
This tool mentor explains how to add comments and annotations on application models.
Tool: IBM Rational Rhapsody - Design Manager

Main Description

The Web client supports the annotating and commenting of design artifacts through modern implementations of web browser technology, without the need to install additional software onto the user's workstation.

With the Web client you can:

  • Comment on design artifacts.
  • Graphically annotate design diagrams.

Creating and updating text comments

You can locate the design element that you want to comment on by:

  • Browsing in the Design Explorer.
  • Viewing a diagram.
  • Selecting Search for comments or Search for elements with comments.
  • Typing a query into the Search window.

Once you have located the design element you want to comment on, click New Comment or Reply to Comment. Each comment contains a subject and the text comment itself.

If you are commenting about design elements shown in a diagram, you can add graphical annotations to support your textual commentary.

Adding graphical annotations to a diagram

  • Click New Comment. The Design Manager graphical annotation tool palette is displayed:
    Click Line to draw a straight line.
  • Click Pencil to draw a free-form line.
  • Click Text to add text.
  • Click Oval, Rectangle or Cloud to draw that kind of shape.
  • After you have finished adding your annotations, click Save.
